### Adding New Videos
To add more Taylor Swift videos, edit the `taylorSwiftVideos` array in the JavaScript section:

```javascript
const taylorSwiftVideos = [
```

Modify the CSS custom properties to change the color scheme:

```css
:root {
    --primary-pink: #ff69b4;

    --accent-pink: #db7093;
```
